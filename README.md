# Float34

34-key hand-held column-staggered split wireless keyboard, derivative of the [Charybdis Nano](https://github.com/grassfedreeve/CharybdisNano). ZMK firmware on Nice!Nano v2.

Download firmware from [Actions](https://github.com/hiCozyty/float34/actions). Double-tap reset to enter bootloader, then copy the `.uf2` onto the board.
